Game Documentation

Game Overview

This game is a stock market simulation where players start with an initial budget and aim to increase their wealth by trading stocks and buying items. The game is set in a dynamic market environment where stock prices fluctuate over time.

Gameplay Instructions

Players navigate the game by interacting with different in-game icons like the phone icon for trading and the shopping icon for purchasing items. The goal is to strategically buy and sell stocks and items to maximize wealth.

Interface Description

The game interface includes a main canvas where players can view their character, budget, and total wealth. Icons are used to access different game functionalities like trading stocks and shopping.

Player Actions and Strategies

Players can buy or sell stocks, view stock market details, buy items, and view their portfolio. Strategic buying and selling based on market trends are key to increasing wealth.

Difficulty Levels

The game offers different difficulty levels (easy, normal, hard) which affect the volatility and complexity of the stock market in the game.

Help and Support

A help section is available for players to get assistance on different topics like buying stocks, selling items, and understanding game mechanics.
